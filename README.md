# Steam Help
This app aims to help people to complete their games returning a youtube search for every missing achievement given a game.


Getting Started
----------

### Prerequisites
* Docker
* Docker-compose*
* Steam API Key (https://steamcommunity.com/dev)

### Installing

    pipenv install

### Built With
* Python 3.7
* Flask

### Tests

    pytest
